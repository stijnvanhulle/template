### :dart: Features

- Nx for monorepo graph
- Turborepo + PNPM support
- TypeScript support
- Linting with ESLint, Prettier and Husky(Git hooks)
- Testing with Jest
- ESM and CommonJS builds with TSUP
- Release with changeset
